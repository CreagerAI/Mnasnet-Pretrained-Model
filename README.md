# Mnasnet-Imagenet1k-Pretrained

## Introduction

The Mnasnet ImageNet-1K pretrained model.The ImageNet-1k is not the standard (ILSRVRC 2012),which extract from imagenet 11k and split it to 95:5 (train:val).

| Model   | Top-1 | Top-5 |
| ------- | ----- | ----- |
| Mnasnet | 0.720 | 0.905 |
